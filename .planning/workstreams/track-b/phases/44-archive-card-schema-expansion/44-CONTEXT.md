# Phase 44 Context

## Phase
44 - Archive Card Schema Expansion

## Primary Track
Track B - 人物构建与管理层

## Secondary Impact
Track C - Discord 交互层

## Goal
Expand long-lived archive schema toward a fuller COC investigator card without mixing campaign-local truth into long-lived identity truth.

## Agreed Scope
- Add long-lived archive fields for:
  - important person
  - significant location
  - treasured possession
  - trait notes
  - scars and injuries
  - phobias and manias
- Keep birthplace/residence/family/education as long-lived fields
- Preserve backward compatibility for older archive payloads

## Explicitly Deferred
- cash
- assets
- carried inventory

## Boundary
Track B owns the long-lived archive schema and semantics. Track C may later adapt command/display behavior, but this phase does not redesign Discord command flow.
