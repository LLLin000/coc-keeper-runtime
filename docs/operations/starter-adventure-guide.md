# Starter Adventure Guide

## Adventure

`starter_crypt` / `灰烬墓穴`

## Premise

一支小队受雇调查山坡古墓附近失踪的守墓人，并找回一枚被盗的封印戒指。

## Recommended Flow

1. `/load_adventure adventure_id:starter_crypt`
2. 用 DM 描述把玩家带到墓园铁门前
3. 让玩家通过自然消息调查入口、拖痕和守夜灯
4. 推进到前室石棺，给出壁画、血迹和残页线索
5. 在封印室触发最终遭遇

## Scene Nodes

### 1. 墓园铁门

- 重点：建立气氛，给出拖痕和熄灭灯火
- 推荐检定：察觉、生存
- 结果：线索指向地下入口

### 2. 前室石棺

- 重点：把戒指徽记、仪式残页、守墓人痕迹拼起来
- 推荐检定：调查、奥秘、宗教
- 结果：玩家知道有人被迫打开封印

### 3. 封印室

- 重点：战斗或谈判
- 推荐：先短对话再开战
- 关键物：封印戒指

## GM Notes

- 这是 starter module，不是完整长模组
- 目标是让你验证：
  - 多人普通消息输入
  - scene mode
  - combat gating
  - diagnostics and persistence

## Suggested NPC Tone

- 守墓人：惊恐、疲惫、知道一半真相
- 盗墓者：贪婪但未完全失控
- 墓穴守卫：沉默、压迫感强
